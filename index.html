how to start

プロンプト1個コピペで始める

セットアップは「プロンプト1個コピペ」。ターミナル操作を覚える必要なし。Claude Code に起動プロンプトを貼るだけで、AI が対話で全部ガイドします。

1

Claude Code をインストール

Anthropic 社が無料提供する AI アシスタント CLI。持ってない方は、まず公式ガイドからインストールします。

→ Claude Code 公式インストールガイド

Mac の場合: brew install --cask claude-code

2

Claude Code を起動

Mac のターミナルアプリで claude と打ってエンター。AI とのチャットが始まります。

3

セットアッププロンプトをコピペ

→ SETUP_PROMPT.md を開きます。

枠内のプロンプト全文をコピー → Claude Code に貼り付け → エンター。あとは AI が対話的にセットアップを進めます。

4

対話しながら進める

AI が API キー取得・テンプレ設定・動作確認まで1ステップずつガイド。

「わからない」「これスキップ」「Slack 連携にしたい」など、何でも相談OK。

what it does

「全部、AIにお願いする」ワークフロー

初期設定も、日々のZoom運用も、Claude Code と対話しながら進める設計です。

1

「明日19時から○○講座のZoom作って」

AIに頼むだけ。Zoom会議自動作成 → 受講者向けチャットアプリ(Discord)に @everyone 告知投稿 → イベント登録 → 5分前リマインダーを予約 → Googleカレンダーに予定追加。
★ Discord 使ってない場合はこの段階を省略 or 他のチャットアプリ(Slack・LINE公式アカウント・Chatwork 等)に拡張可能

Zoom API チャット連携 Google Calendar launchd
2

配信5分前、リマインダー自動送信

「🟢 この後19:00から○○講座を開始します」が受講者チャット(Discord等)に自動投稿。@everyone 通知で受講生に確実に届く。Mac を触らなくても launchd cron が裏で動く。

launchd Webhook
3

翌朝7:30、録画フル自動処理

Zoomクラウドから録画DL → Claude Haiku で要約生成(主な内容/キーワード/持ち帰りポイント)→ Google Drive にバックアップ → YouTube 限定公開アップロード → 受講者チャットのアーカイブチャンネルに視聴URL投稿 → Obsidian Vault に整形版保存。

Zoom API Claude Haiku Google Drive YouTube API Obsidian
4

あなたが朝起きたら、すべて完了している

受講生は Discord でアーカイブ URL を受け取って復習開始。あなたは Obsidian で要約をチェックして次の配信準備に集中。手作業ゼロ。

📚

🤖

ターミナル操作やAPI設定に不安がある。手順書を読むより、AIに聞きながら進めたい方

🍵

「使い方」より「中身」

features

機能ハイライト

セットアップはプロンプト1個。運用は「Zoom作って」「録画処理して」と頼むだけ。

🎯

テンプレで講座を分類

「メインコース」「サブコース」「定例会」「個別セッション」など、自分の講座構造を JSON で自由に定義。トピック名で投稿先・YouTube アップ要否を自動判定。

📢

チャットアプリ 3点セット連動

事前告知(@everyone)+ 5分前リマインダー + アーカイブURL案内 を、受講者向けチャットアプリ(Discord等)のサーバー&チャンネル別に自動振り分け投稿。

📅

イベント自動登録

チャットアプリの「イベント」機能(Discord Scheduled Events)にも自動登録。受講生は「興味あり」を押せば開始時に通知が届く。

🤖

Claude Haiku で要約生成

OpenRouter 経由で動画1本約2円。一言サマリー / 主な内容 / キーワード / 持ち帰りポイントを構造化生成。

📺

YouTube 限定公開UP

speaker_view を限定公開でアップ。タイトル・説明・タグも要約から自動生成。個別面談系は自動除外。

📓

Obsidian 整形版保存

Vault の「文字起こし/Zoom/」に frontmatter + 要約 + 動画リンク3点を整形保存。後から検索・参照しやすく。

☁️

Google Drive バックアップ

動画 + 字幕 + 要約 + メタ情報をフォルダごと自動バックアップ。冪等性あり、何度実行しても重複しない。

⏰

launchd で完全自動

毎朝7:30に未処理録画をフル自動処理。caffeinate ラップ済みでスリープ中も安全に完走。

🛡️

個別面談は自動除外

「面談 / コーチング / パーソナル / 1on1」キーワード入りトピックは YouTube アップから自動除外。プライバシー安全。

🔧

使ってない機能は省略OK

Discord / Obsidian / YouTube はどれも任意。--no-discord --no-obsidian --no-youtube オプションで個別スキップ可能。Slack や LINE 公式アカウント、Chatwork など Webhook 対応の他チャットアプリへの拡張も同じ仕組みで可能。
