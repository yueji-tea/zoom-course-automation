# 01. このパッケージは何?

## このセクションの目的

このドキュメントを読むと、「zoom-course-automation」というパッケージが **何を自動でやってくれるのか**、そして **自分の講座運営でどう役立つのか** が分かるようになります。

技術初心者でも導入できるように設計されています。プログラミングの知識は必要ありません(コピー&ペーストでターミナルにコマンドを貼れれば大丈夫です)。

## 所要時間

5分(読むだけ)

## 前提

特になし

---

## このパッケージで実現される自動化フロー

オンライン講座(Zoom開催)を運営していると、毎回こんな作業が発生します。

1. Zoom会議を作る
2. 受講者に「明日◯時から開講します」と告知する
3. 開始5分前にリマインダーを送る
4. 録画ができたらダウンロードする
5. 録画の内容を要約する(復習用に)
6. Google Drive にバックアップする
7. YouTube に限定公開でアップする(欠席者向け)
8. Discordサーバーにアーカイブ案内を投稿する
9. Obsidian(メモアプリ)に記録を残す

このパッケージを入れると、上の **1〜9 がほぼ全自動** になります。

ゆえじが運営する茶養講座・ダブハピ団・朝茶会などで実際に使っているフローを、誰でも使えるようにパッケージ化したものです。

### ざっくり全体図

```
[Zoom作成]
   ↓
[Discord告知 + Eventsへ登録 + 5分前リマインダー予約 + Googleカレンダー追加]
   ↓
(講座当日)
   ↓
[録画完了]
   ↓
[録画一覧取得] → [DL] → [AIで要約生成] → [Drive バックアップ]
                                          ↓
                                  [YouTube限定公開アップ]
                                          ↓
                              [Discordへアーカイブ案内投稿]
                                          ↓
                              [Obsidianに記録保存]
```

すべて1コマンドで動きます。

---

## ユースケース例

このパッケージは、こんな方に向いています。

- **オンライン講座運営者**:Zoomで定期講座を開いていて、毎回の事前準備・事後処理に時間を取られている方
- **コーチ / カウンセラー**:1対1セッションやグループセッションの録画管理を楽にしたい方
- **コミュニティ運営者**:Discord等で運営しているコミュニティで、定例会の告知〜アーカイブまでを自動化したい方
- **書籍出版準備中の方**:過去のセッション録画を一気に処理して、原稿の素材集めをしたい方

特に「個別面談・コーチング・パーソナル系」の録画は YouTubeアップロードから自動除外する設定が入っているので、プライバシーへの配慮も組み込み済みです。

## 必要な月額コスト目安

| サービス | 月額目安 | 必須? |
|---|---|---|
| **OpenRouter**(AI要約用) | $5 程度 (動画1本数円〜数十円) | 必須 |
| **Zoom Pro** 以上 | 既に契約済みの想定 | 必須(クラウド録画機能のため) |
| **Google Workspace**(Drive用) | 無料枠で足りる場合が多い | 推奨 |
| **YouTube** | 無料 | 任意 |
| **Discord** | 無料 | 任意 |
| **Obsidian** | 無料(Sync使うなら有料) | 任意 |

合計:**だいたい 月 $5〜$10**(700円〜1500円)で運用できます。

OpenRouter の代わりに Anthropic 公式 API を使えば、さらに5%ほど安くなります(後述)。
